# Contributing

Thank you for your interest in contributing to this project. Here's how you can help.

---

## Ways to Contribute

### 1. Report Your Results

The most valuable contribution is telling us whether the guide worked for you.

**If it worked:** Add your device to [TESTED-DEVICES.md](TESTED-DEVICES.md)

**If it failed:** Open an issue with details about what went wrong

### 2. Improve Documentation

Found a typo? Something unclear? A step that could be explained better?

- Fork the repository
- Make your changes
- Submit a pull request

### 3. Add Troubleshooting Solutions

Encountered a problem and solved it? Add your solution to [TROUBLESHOOTING.md](TROUBLESHOOTING.md)

### 4. Translate

Want to translate this guide into another language? Translations are welcome. Create a new file like `README.es.md` (Spanish), `README.de.md` (German), etc.

---

## Reporting Issues

When opening an issue, please include:

1. **Mac model and year**
2. **Chip** (M1, M2, M3, M4, etc.)
3. **MDM provider** (if known)
4. **Which step failed**
5. **Exact error message** (copy/paste, not paraphrase)
6. **What you already tried**
7. **Linux distro** you're using for the restore process

The more detail you provide, the better we can help.

---

## Pull Request Guidelines

- Keep changes focused - one fix/improvement per PR
- Test your changes if possible
- Update relevant documentation (README, TROUBLESHOOTING, etc.)
- Use clear commit messages

---

## Code of Conduct

- Be respectful
- Be helpful
- Remember that everyone is here because they're frustrated with a locked device - patience goes a long way

---

## What We Don't Accept

- Scripts or tools that automate the bypass (this keeps the project as documentation only)
- Anything that could be construed as a "hacking tool"
- Content that encourages illegal use

---

## Questions?

Open an issue with the "question" label.
