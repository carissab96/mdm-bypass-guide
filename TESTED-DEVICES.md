## How to Contribute

### If successful:

Please provide:
- Mac model and year
- Chip (M1, M1 Pro, M1 Max, M1 Ultra, M2, M2 Pro, M2 Max, M3, M3 Pro, M3 Max, M4, etc.)
- MDM provider (if known) - Mosyle, Jamf, Kandji, etc.
- macOS version you restored to
- Asahi/Linux version you installed
- Any notes about issues you encountered and solved

### If failed:

Please provide:
- Mac model and year
- Chip
- MDM provider (if known)
- Which step failed
- Exact error message or description of what went wrong
- Any troubleshooting you attempted
